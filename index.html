Pretty-Install
by Briks Software

Use a doorhanger notification for add-on installation in Firefox

There was a plan to turn the Firefox add-on install dialog into a doorhanger dialog (bug 588266). This was abandoned in favour of an approach using tabs (bug 643020). Rather than let the work for the former approach go to waste, we are releasing it as an add-on.

Install

Run the 'build.sh' shell script and install the generated XPI file in Firefox.
